# Aivo-v2 数字人智能体 V4

> 基于小米MiMo API的智能数字人系统 - **精美卡通 | 口型同步 | 9种工具 | 8种音色**

## ✨ V4 最新特性

- 🎨 **精美卡通数字人** - 渐变着色，完整五官，立体感强
- 👄 **真实口型同步** - 跟随音频播放，自动张嘴闭嘴
- 🛠️ **9个实用工具** - 天气、时间、计算、搜索、提醒、翻译、维基、笔记、新闻
- 🎵 **8种音色选择** - 中英文男女声，自由切换
- 🎤 **语音输入输出** - 完整的语音交互体验
- 🌐 **现代化Web界面** - Gradio 6.0，渐变设计

## 🎉 核心功能

- ✅ **语音识别（ASR）** - MiMo API，支持中英文
- ✅ **智能对话（LLM）** - MiMo mimo-v2.5-pro，1T参数
- ✅ **语音合成（TTS）** - MiMo API，8种音色+情感控制
- ✅ **精美2D动画** - 卡通风格，真实口型同步
- ✅ **工具调用系统** - 9个工具，自动识别意图
- ✅ **多轮对话** - 上下文记忆
- ✅ **本地部署** - 无需GPU，Mac本地运行
- ✅ **Web界面** - 美观易用的交互界面

## 🚀 快速开始

### 1. 安装依赖

```bash
bash install.sh
# 或手动安装
pip install openai python-dotenv pyaudio requests gradio
```

### 2. 配置API密钥

```bash
cp .env.example .env
vim .env  # 填入MIMO_API_KEY
```

### 3. 启动Web界面（推荐）

```bash
bash run.sh
# 选择选项 1 - Web界面V3
```

或直接运行：
```bash
python src/web_ui_v3.py
```

浏览器会自动打开 `http://localhost:7860`

### 4. 其他运行方式

**文本对话模式**：
```bash
python src/main_v2.py --mode text
```

**语音对话模式**：
```bash
python src/main_v2.py --mode voice --duration 5
```

**测试工具**：
```bash
bash run.sh  # 选择选项 6
```

## 🛠️ 核心模块

### ASR服务 (`asr_service.py`)
- 录音功能（使用pyaudio）
- 调用MiMo ASR API
- 支持中英文识别

### LLM服务 (`llm_service_v2.py`)
- 多轮对话管理
- 动作指令协议输出
- **工具调用支持**（Function Calling）
- 自动识别需要调用的工具
- 工具结果整合到回复中

### TTS服务 (`tts_service.py`)
- 文本转语音
- 8种音色智能选择
- 情感标签自动添加
- 音频播放

### 工具系统 (`tools/`) - 9个工具

**已实现工具：**
1. 🌤️ **天气查询** - 实时天气查询（wttr.in API）
2. ⏰ **时间日期** - 获取当前时间和日期
3. 🔢 **计算器** - 数学运算
4. 🔍 **网络搜索** - DuckDuckGo搜索
5. ⏱️ **提醒工具** - 设置和管理提醒事项
6. 🌐 **翻译工具** - 中英互译（MyMemory API）
7. 📚 **维基百科** - 知识查询
8. 📝 **笔记工具** - 本地笔记管理
9. 📰 **新闻工具** - 获取资讯

## 🎨 音色配置

支持8种音色，根据情感自动选择：
- **中文女声**: 冰糖（活泼）、茉莉（温柔）
- **中文男声**: 苏打（沉稳）、白桦（阳光）
- **英文女声**: Mia、Chloe
- **英文男声**: Milo、Dean

情感自动映射：
- `happy` → 冰糖/白桦（明亮）
- `sad` → 茉莉/苏打（温柔）
- `neutral` → 茉莉/苏打（平和）
- `surprised` → 冰糖/白桦（有活力）
