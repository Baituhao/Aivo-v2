# Aivo V6 - 智能数字人系统

> **版本**: V6.0 | **最后更新**: 2026-06-21 | **状态**: 生产就绪

基于小米 MiMo API 的智能数字人系统，集成 **Live2D 数字人**、**多模态交互**、**会话管理**和 **Soul Memory 灵魂记忆系统**。

**GitHub**: https://github.com/Baituhao/Aivo-v2

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🎀 **Live2D 数字人** | 3 个模型可切换，语音同步动画 |
| 🎵 **气泡语音** | 音频嵌入对话气泡，自动播放 |
| 🛠️ **9 种工具** | 天气、搜索、翻译、计算、新闻等 |
| 🎤 **8 种音色** | 中英文男女声，情感自动匹配 |
| 📊 **实时统计** | 对话轮次、响应时间、情感分布 |
| 📜 **会话管理** | 持久化存储、多会话切换、跨会话搜索 |
| 🧠 **Soul Memory** | 人格设定、用户画像、技能管理 |

## 🚀 快速开始

### 1. 安装依赖

```bash
# 使用 conda（推荐）
conda create -n aivo python=3.10
conda activate aivo
pip install -r requirements.txt

# 或使用 pip
pip install openai python-dotenv pyaudio requests gradio pillow
```

### 2. 配置 API 密钥

```bash
cp .env.example .env
# 编辑 .env 文件，填入 MIMO_API_KEY
```

### 3. 启动服务

```bash
python src/web_ui_v6.py
```

浏览器自动打开 `http://localhost:7860`

## 🎯 功能详解

### 💬 智能对话

基于 MiMo 大模型的多轮对话，支持：
- 上下文记忆（最近 10 轮）
- 结构化响应（speech/emotion/action）
- 工具调用自动识别

### 🎀 Live2D 数字人

基于 Cubism SDK 4 的 Live2D 渲染：
- **3 个模型**: Shizuku、Hijiki、Tororo
- **语音同步**: 说话时嘴型自动匹配
- **情感表情**: 根据情感自动切换
- **点击交互**: 点击切换模型

### 🛠️ 工具系统

| 工具 | 功能 |
|------|------|
| 🌤️ 天气查询 | 实时天气信息 |
| ⏰ 时间日期 | 当前时间日期 |
| 🔢 计算器 | 数学运算 |
| 🔍 网络搜索 | 信息检索 |
| ⏱️ 提醒工具 | 设置提醒事项 |
| 🌐 翻译工具 | 中英互译 |
| 📚 维基百科 | 知识查询 |
| 📝 笔记工具 | 本地笔记管理 |
| 📰 新闻工具 | 获取资讯 |

### 🎤 语音交互

**8 种音色**:
- **中文女声**: 冰糖（活泼）、茉莉（温柔）
- **中文男声**: 苏打（沉稳）、白桦（阳光）
- **英文女声**: Mia、Chloe
- **英文男声**: Milo、Dean

**情感音色映射**:
- `happy` → 冰糖/白桦
- `sad` → 茉莉/苏打
- `neutral` → 茉莉/苏打
- `surprised` → 冰糖/白桦

### 📊 实时统计面板

- 💬 对话轮次
- ⚡ 平均响应时间
- 🎭 情感切换次数
- 📝 总字数
- ⏱️ 会话时长
- 🔧 工具调用次数
- 情感分布可视化图表

### 📜 会话管理

- ✅ 自动保存会话到 `sessions/` 目录
- ✅ 启动时自动加载最近会话
- ✅ 新建会话、加载历史会话
- ✅ 对话框左上角快速切换
- ✅ 会话选择器显示第一条消息
- ✅ 导出会话为 JSON 文件
- ✅ **跨会话搜索** - 在所有会话中搜索关键词

### 🧠 Soul Memory 系统

**人格设定**:
- 名称、角色、特质、说话风格

**用户画像**:
- 兴趣爱好、使用习惯、背景信息

**技能系统**:
- 知识领域、自定义技能、等级管理

**偏好设置**:
- 语言、回复风格、表情使用、正式程度

## 🛠️ 核心模块

### ASR 服务 (`asr_service.py`)
- 录音功能（PyAudio）
- 调用 MiMo ASR API
- 支持中英文识别

### LLM 服务 (`llm_service_v2.py`)
- 多轮对话管理
- 工具调用支持（Function Calling）
- 会话历史恢复

### TTS 服务 (`tts_service.py`)
- 文本转语音
- 8 种音色智能选择
- 情感标签自动添加

### 工具系统 (`tools/`)
- 9 个实用工具
- 自动意图识别
- 统一注册管理

## 🚀 使用示例

### 基础对话
```
你：你好，我是小明
Aivo：你好小明！很高兴认识你！ [😊 happy]
```

你：北京今天天气怎么样？
Aivo：[调用天气工具] 北京今天晴，温度 25 度，很适合出门呢！
