# Aivo V6 - 完整功能列表

## 🎉 版本亮点

Aivo V6 是一个功能完整的 AI 数字人系统，集成了 Live2D、多模态交互、会话管理和灵魂记忆系统。

---

## ✨ 核心功能

### 1. 💬 智能对话系统
- **多模态交互**：文本输入 + 语音输入
- **情感识别**：自动分析并匹配情感表达
- **工具调用**：集成 9+ 种实用工具
- **上下文记忆**：保持对话连贯性

### 2. 🎀 Live2D 数字人
- **实时表情**：根据情感自动切换表情动画
- **模型切换**：3 个内置模型（shizuku, hijiki, tororo）
- **语音同步**：说话时嘴型自动匹配
- **交互式操作**：点击切换模型

### 3. 🎵 语音交互
#### TTS (文字转语音)
- **8 种音色**：冰糖、茉莉、苏打、白桦、Mia、Chloe、Milo、Dean
- **情感表达**：happy, sad, neutral, angry, surprised
- **气泡内嵌**：音频直接嵌入对话气泡，自动播放
- **中英双语**：支持中文和英文音色

#### ASR (语音识别)
- **麦克风录音**：直接语音输入
- **文件上传**：支持音频文件识别
- **实时转写**：语音转文字显示

### 4. 🛠️ 工具生态系统
- 🌤️ **天气查询**：实时天气信息
- 🔍 **网络搜索**：信息检索
- 🌐 **文本翻译**：多语言翻译
- 🧮 **数学计算**：复杂计算
- 📅 **日期时间**：时间查询
- 📰 **新闻资讯**：最新新闻
- 💱 **汇率换算**：货币转换
- 🎲 **随机数生成**：随机数工具
- 📝 **文本处理**：文本操作
- 🎨 **创意灵感**：创意建议

### 5. 📊 实时统计面板
#### 数据指标
- 💬 **对话轮次**：总对话次数
- ⚡ **平均响应时间**：性能监控
- 🎭 **情感切换次数**：情感变化统计
- 📝 **总字数**：累计输出字数
- ⏱️ **会话时长**：当前会话时间
- 🔧 **工具调用次数**：工具使用统计

#### 可视化
- **情感分布图**：柱状图显示各情感占比
- **实时更新**：数据实时刷新
- **美观展示**：渐变卡片设计

### 6. 📜 会话管理系统
#### 自动持久化
- **自动保存**：每次对话自动保存到 `sessions/` 目录
- **会话恢复**：启动时自动加载最近会话
- **会话历史**：保存所有历史会话

#### 会话操作
- 🆕 **新建会话**：保存当前会话，创建新对话
- 🗑️ **清空所有**：保存后重置所有数据
- 📂 **加载会话**：从历史中恢复任意会话
- 💾 **导出会话**：导出为 JSON 文件
- 🔍 **搜索对话**：关键词搜索历史内容

#### 会话数据
```json
{
  "session_id": "20250621_143000",
  "created_at": "2025-06-21 14:30:00",
  "last_updated": "2025-06-21 15:45:00",
  "conversation_history": [...],
  "stats": {...}
}
```

### 7. 🧠 Soul Memory 系统
#### 人格设定 (🎭)
- **名称**：AI 名字
- **角色**：AI 定位
- **特质**：性格特点（多个标签）
- **说话风格**：表达方式

#### 用户画像 (👤)
- **用户名称**：个性化称呼
- **兴趣爱好**：话题偏好
- **使用习惯**：行为模式
- **背景信息**：职业/专业等

#### 技能系统 (🛠️)
- **知识领域**：5 个默认领域
- **自定义技能**：可添加任意技能
- **技能等级**：初级/中级/专家
- **启用控制**：开启/关闭特定技能

#### 偏好设置 (⚙️)
- **语言**：中文/英文/日语
- **回复风格**：简洁/适中/详细
- **表情使用**：少/适中/多
- **正式程度**：随意/友好/正式

#### 记忆持久化
- 存储位置：`memory/` 目录
- 自动加载：启动时加载
- 导入导出：支持备份迁移

### 8. 💡 快速示例系统
#### 一键发送
- 点击示例按钮**立即发送**，无需手动输入
- 8 个预设示例：
  - 🌤️ 北京天气
  - 🔍 AI最新进展
  - 📰 科技新闻
  - 🌐 翻译Hello
  - 📅 今天星期几
  - 💬 讲个笑话
  - 🧮 1+1等于几
  - 🎵 推荐歌曲

#### 工具快捷填充
- 10 个工具按钮
- 点击填充到输入框
- 可修改后发送

## 📂 项目结构

```
Aivo-v2/
├── src/
│   ├── web_ui_v6.py          # 主界面（核心）
│   ├── soul_memory.py         # 灵魂记忆系统
│   ├── llm_service_v2.py      # LLM 服务
│   ├── tts_service.py         # TTS 服务
│   ├── asr_service.py         # ASR 服务
│   ├── voice_manager.py       # 音色管理
│   └── static/
│       ├── live2d_pixi.html   # Live2D 页面
│       ├── live2dcubismcore.min.js
│       └── pixi-live2d-display.min.js
├── memory/                    # Soul Memory 存储
│   ├── personality.json
│   ├── user_profile.json
│   ├── skills.json
│   └── preferences.json
├── sessions/                  # 会话历史存储
│   └── session_*.json
├── config/
│   └── voice_settings.json    # 音色配置
├── requirements.txt           # Python 依赖
├── SOUL_MEMORY_GUIDE.md      # Soul Memory 使用指南
└── README.md
```

浏览器自动打开 http://localhost:7860

## 📈 技术栈

- **前端框架**：Gradio
- **AI 模型**：Claude 4 (LLM)
- **语音合成**：Fish Audio TTS
- **语音识别**：ASR Service
- **数字人**：Live2D (Cubism SDK 4 + PixiJS)
- **数据存储**：JSON 文件
- **编程语言**：Python 3.8+
