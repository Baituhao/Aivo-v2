# Aivo-v2 最终项目报告

## 项目概述

Aivo-v2 是一个基于小米MiMo API的智能数字人系统，支持实时对话、工具调用、情感表达和多轮交互。

## 完成情况

### ✅ 已实现功能

#### 1. 核心对话系统
- [x] ASR语音识别（MiMo API）
- [x] LLM对话大脑（mimo-v2.5-pro，1T参数）
- [x] TTS语音合成（8种音色+情感控制）
- [x] 多轮对话上下文记忆（保留最近10轮）
- [x] 动作指令协议（JSON格式）

#### 2. 工具调用系统

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Gradio 6.0 | 现代化Web界面 |
| **后端** | Python 3.8+ | 核心逻辑 |
| **LLM** | MiMo mimo-v2.5-pro | 1T参数，1M上下文 |
| **ASR** | MiMo ASR API | 中英文识别 |
| **TTS** | MiMo TTS API | 8种音色 |
| **工具** | wttr.in, DuckDuckGo | 免费API |

## 使用指南

### 快速启动

```bash
# 1. 安装依赖
bash install.sh

# 2. 配置API密钥
vim .env  # 填入MIMO_API_KEY

# 3. 启动Web界面
bash run.sh
# 选择 1 (Web界面)

# 或直接运行
python src/web_ui.py
```

### 运行测试

```bash
# 工具测试
python tests/test_tools.py

# 性能测试
python scripts/performance_test.py

# Demo演示
python scripts/demo.py
```
