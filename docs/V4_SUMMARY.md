### 口型同步原理

```javascript
// 核心代码（已内置在HTML中）

// 1. 监听audio播放
audio.addEventListener('play', () => {
    startMouthAnimation();  // 开始张嘴
});

// 2. 监听停止
audio.addEventListener('ended', () => {
    stopMouthAnimation();  // 闭嘴
});

// 3. 口型动画循环
setInterval(() => {
    isOpen = !isOpen;
    mouthPath.setAttribute('d',
        isOpen ? openPath : closePath
    );
}, 150);
```
