## ⚠️ 注意事项

### 必需依赖
确保安装了以下依赖：
```bash
pip install openai python-dotenv pyaudio requests gradio pillow
```

或使用：
```bash
bash install.sh
```

### API配置
确保`.env`文件配置正确：
```
MIMO_API_KEY=your_key_here
MIMO_API_BASE=https://token-plan-cn.xiaomimimo.com/v1
```

### 网络要求
- 需要稳定的网络连接（调用MiMo API）
- 翻译和维基百科功能需要外网访问
