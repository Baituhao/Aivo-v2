# Aivo V3 快速使用指南

## 🚀 快速开始

### 1. 启动系统

```bash
bash run.sh
```

选择 **选项1** - Web界面V3（完整版）

浏览器会自动打开 `http://localhost:7860`

### 2. 界面说明

#### 左侧 - 对话区
- **对话历史**：显示你和Aivo的聊天记录
- **文本输入框**：输入消息
- **语音输入区**：录音或上传音频文件
- **识别结果**：语音识别的文本
- **语音输出**：Aivo的语音回复（自动播放）

#### 右侧 - 数字人区
- **2D动画头像**：实时显示表情和口型
- **统计面板**：对话轮数、工具调用、情感切换
- **音色选择**：8种音色可选
- **工具列表**：查看9个可用工具
- **快速示例**：点击即可发送示例问题

## 🎵 音色选择

**中文女声**：
- **冰糖** - 活泼开朗，适合日常对话
- **茉莉** - 温柔优雅，适合安慰劝导

**中文男声**：

**英文音色**：
- Mia / Chloe（女声）
- Milo / Dean（男声）

选择后即刻生效，下次对话会使用新音色。

## 🎤 语音输入

### 方法1：浏览器录音
1. 点击"🎤 语音输入"区域
2. 允许浏览器使用麦克风
3. 点击录音按钮
4. 说话（3-5秒）
5. 停止录音
6. 点击"🎙️ 识别语音"

### 方法2：上传音频
1. 点击"上传"标签
2. 选择音频文件（.wav/.mp3）
3. 点击"🎙️ 识别语音"

识别结果会显示在"识别结果"框中，Aivo会自动回复。

## 🤖 数字人动画

头像会根据情感自动变化：

| 情感 | 表情 | 触发场景 |
|------|------|----------|
| 😊 开心 | 笑眯眯的眼睛 + 笑脸 | "太好了"、"成功" |
| 😢 难过 | 下垂的眼睛 + 皱眉 | "失败"、"可惜" |
| 😐 平静 | 平直的眼睛 + 直线嘴 | 中性对话 |
| 😲 惊讶 | 圆眼睛 + O型嘴 | "什么"、"真的吗" |

说话时：
- 嘴巴会开合（口型动画）
- 显示声波效果

## 🎯 使用技巧

### 1. 多步骤任务
```
你："帮我安排一下明天的活动"
Aivo：会自动：
  1. 调用时间工具获取日期
  2. 调用天气工具查询天气
  3. 给出建议
```

### 2. 连续对话
```
你："北京今天天气怎么样？"
Aivo："北京今天晴，29度"

你："那上海呢？"  # Aivo会理解上下文
Aivo：调用天气工具查询上海
```

### 3. 清空历史
点击"🗑️ 清空历史"可以：
- 清空对话记录
- 重置统计数据
- 回到初始状态

## ⚙️ 常见问题

### Q1: 语音识别失败？
**A**:
- 检查麦克风权限
- 确保录音环境安静
- 说话清晰（3-5秒）

### Q2: 工具调用没反应？
**A**:
- 查看控制台日志
- 确认问题表述清晰
- 检查网络连接

### Q3: 音色切换无效？
**A**:
- 下次对话会生效
- 检查音色选择器

### Q4: 数字人不动？
**A**:
- 刷新页面
- 检查浏览器是否支持SVG
