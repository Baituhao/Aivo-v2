### 完整数据流

```
Mac 本地完整链路（0显存，无需DSW）

用户说话
  ↓
录音（Mac麦克风）
  ↓
MiMo ASR API → 文本
  ↓
MiMo LLM (mimo-v2.5-pro)
  ├─ 意图识别
  ├─ 工具调用（天气/搜索/数据库）
  └─ 生成JSON：{"speech": "...", "emotion": "happy", "action": "nod"}
  ↓
MiMo TTS API → 语音（带情感）
  ↓
Mac前端
  ├─ 播放音频
  ├─ 显示文字
  ├─ 切换表情图片（根据emotion）
  └─ 播放动作动画（根据action）
```
