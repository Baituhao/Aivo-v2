## 情感标签支持

### 基础情感
- `[开心]` - 愉悦、兴奋
- `[悲伤]` - 低落、难过
- `[兴奋]` - 激动、惊喜
- `[平静]` - 淡定、从容
- `[紧张]` - 焦虑、不安
- `[愤怒]` - 生气、不满

### 高级情感（用圆括号）
- `(怅然)` - 惆怅、若有所失
- `(欣喜)` - 由衷的高兴
- `(惊讶)` - 意外、吃惊
- `(温柔)` - 柔和、体贴

### 其他控制标签
- `[呼吸]` - 添加呼吸音
- `[停顿]` - 短暂停顿
- `[加快]` - 语速加快
- `[放慢]` - 语速放慢

## 配置示例

### 场景1：日常闲聊
```python
voice = "冰糖"  # 明亮活泼的女声
text = "[开心]早上好！今天天气真不错呢！"
```

### 场景2：安慰用户
```python
voice = "茉莉"  # 温柔平静的女声
text = "(温柔)别担心，一切都会好起来的。[停顿]你已经做得很好了。"
```

### 场景3：播报天气
```python
voice = "苏打"  # 沉稳的男声
text = "[平静]今天北京晴，气温25度，适合户外活动。"
```

### 场景4：兴奋通知
```python
voice = "冰糖"  # 活泼的女声
text = "[兴奋]太好了！[停顿]你的任务已经完成啦！"
```

## 动态音色选择策略

根据对话内容和情感自动选择音色：

```python
def select_voice(emotion: str, gender_preference: str = "female"):
    """根据情感和性别偏好选择音色"""

    voice_map = {
        "female": {
            "happy": "冰糖",      # 开心用活泼的
            "sad": "茉莉",        # 悲伤用温柔的
            "neutral": "茉莉",    # 中性用优雅的
            "surprised": "冰糖",  # 惊讶用明亮的
        },
        "male": {
            "happy": "白桦",      # 开心用阳光的
            "sad": "苏打",        # 悲伤用沉稳的
            "neutral": "苏打",    # 中性用大气的
            "surprised": "白桦",  # 惊讶用亲和的
        }
    }

    return voice_map.get(gender_preference, {}).get(emotion, "冰糖")
```
